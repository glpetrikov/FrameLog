# Documentation

## Tokens

1. #y12 = 25                    Year2
2. #y24 = 2025                  Year4
3. #message = UserMessage       Message  
4. #m = 10                      Month
5. #d = 20                      Day
6. #h12 = 4                     Hour12
7. #h24 = 16                    Hour24
8. #M = 05                      Minute
9. #s = 01                      Second
10. #Ms                         MilliSeconds
